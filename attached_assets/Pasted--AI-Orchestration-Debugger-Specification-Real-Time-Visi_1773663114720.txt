# AI Orchestration Debugger Specification
## Real-Time Visibility into AI Agent Processing & Prompt Optimization

**Project**: Lendry Lending Application
**Purpose**: Enable real-time visibility into AI orchestration layer for prompt optimization and debugging
**Target Audience**: Developers during build phase, Admins in production (with feature flag)
**Tech Stack**: React 18 + TypeScript, OpenAI API, Service class orchestrator pattern, WebSocket streaming

---

## 1. System Architecture Overview

### 1.1 High-Level Flow

```
User Action (Upload Credit Policy / Run AI Orchestration)
    ↓
Orchestrator Service (orchestrator.ts) - Instrumented with events
    ↓
Tracing/Instrumentation Layer - Captures agent state, inputs, outputs
    ↓
WebSocket Server - Streams real-time events to frontend
    ↓
React Sidebar Component (AIOrchestrationDebugger) - Receives and displays data
    ↓
Developer sees: [Agent Name] → [Input] → [Processing] → [Output]
Optional: Adjust prompt → Replay isolated agent or cascade
```

### 1.2 Key Components

**Backend:**
- `orchestrator.ts` (existing service) - Add instrumentation hooks
- `services/orchestrationTracing.ts` (new) - Capture agent execution events
- `websocket/orchestrationEvents.ts` (new) - Stream events to connected clients
- `middleware/debuggerAuth.ts` (new) - Protect debugger endpoints

**Frontend:**
- `components/AIOrchestrationDebugger/` (new)
  - `DebuggerSidebar.tsx` - Main container
  - `AgentTracePanel.tsx` - Shows agent execution flow
  - `RealTimeOutput.tsx` - Streams agent processing
  - `PromptEditor.tsx` - Edit and replay prompts
  - `CreditExtractionPreview.tsx` - Credit policy extraction preview
- `hooks/useOrchestrationEvents.ts` (new) - WebSocket consumer hook
- `types/orchestration.ts` (new) - TypeScript interfaces

## 2. Backend Implementation

### 2.1 Instrumentation Layer (`services/orchestrationTracing.ts`)

**Purpose**: Intercept agent calls and capture execution data without modifying orchestrator logic.

**Key Events to Capture:**
```typescript
interface OrchestrationEvent {
  timestamp: ISO8601;
  sessionId: string;           // Unique session for this orchestration run
  eventType: 'agent_start' | 'agent_processing' | 'agent_complete' | 'agent_error';
  agentName: string;           // "creditExtractor" | "dealAnalyzer" | etc.
  agentIndex: number;          // Position in chain (1, 2, 3, 4)
  input: Record<string, any>;  // What the agent received
  output?: Record<string, any>; // What the agent produced
  rawResponse?: string;        // Raw AI response before parsing
  parsedOutput?: any;          // Parsed/extracted output
  prompt?: string;             // The system prompt used
  tokens?: { input: number; output: number };
  duration?: number;           // ms to complete
  error?: string;              // If failed
  metadata?: {
    model: string;            // "gpt-4", "gpt-4-turbo", etc.
    temperature?: number;
    tokens_used?: number;
  };
}
```

**Implementation Pattern:**

```typescript
// services/orchestrationTracing.ts
export class OrchestrationTracer {
  private static subscribers: Set<(event: OrchestrationEvent) => void> = new Set();
  private static currentSessionId: string;

  static startSession(): string {
    this.currentSessionId = generateUUID();
    return this.currentSessionId;
  }

  static subscribe(callback: (event: OrchestrationEvent) => void): () => void {
    this.subscribers.add(callback);
    return () => this.subscribers.delete(callback);
  }

  static async traceAgent<T>(
    agentName: string,
    agentIndex: number,
    input: Record<string, any>,
    agentFn: () => Promise<T>,
    prompt?: string
  ): Promise<T> {
    const startTime = Date.now();

    // Agent start event
    this.emit({
      eventType: 'agent_start',
      agentName,
      agentIndex,
      input,
      prompt,
      timestamp: new Date().toISOString(),
      sessionId: this.currentSessionId,
    });

    try {
      const result = await agentFn();

      // Agent complete event
      this.emit({
        eventType: 'agent_complete',
        agentName,
        agentIndex,
        input,
        output: result,
        duration: Date.now() - startTime,
        timestamp: new Date().toISOString(),
        sessionId: this.currentSessionId,
      });

      return result;
    } catch (error) {
      // Agent error event
      this.emit({
        eventType: 'agent_error',
        agentName,
        agentIndex,
        error: error.message,
        duration: Date.now() - startTime,
        timestamp: new Date().toISOString(),
        sessionId: this.currentSessionId,
      });
      throw error;
    }
  }

  private static emit(event: OrchestrationEvent) {
    this.subscribers.forEach(cb => cb(event));
  }
}
```

**Integration into Orchestrator:**

Modify `orchestrator.ts` to wrap each agent call:

```typescript
// In orchestrator.ts (Orchestrator service class)
async executeOrchestration(input: Record<string, any>) {
  const sessionId = OrchestrationTracer.startSession();

  // Agent 1: Credit Extraction
  const creditRules = await OrchestrationTracer.traceAgent(
    'creditExtractor',
    1,
    input,
    () => this.creditExtractionAgent(input),
    CREDIT_EXTRACTION_PROMPT // Pass current prompt
  );

  // Agent 2: Deal Analysis
  const dealAnalysis = await OrchestrationTracer.traceAgent(
    'dealAnalyzer',
    2,
    { creditRules, ...input },
    () => this.dealAnalysisAgent(creditRules),
    DEAL_ANALYSIS_PROMPT
  );

  // Agent 3: Risk Assessment
  const riskAssessment = await OrchestrationTracer.traceAgent(
    'riskAssessor',
    3,
    { dealAnalysis, creditRules, ...input },
    () => this.riskAssessmentAgent(dealAnalysis),
    RISK_ASSESSMENT_PROMPT
  );

  // Agent 4: Recommendation Engine
  const recommendation = await OrchestrationTracer.traceAgent(
    'recommendationEngine',
    4,
    { riskAssessment, dealAnalysis, creditRules, ...input },
    () => this.recommendationEngine(riskAssessment),
    RECOMMENDATION_PROMPT
  );

  return { creditRules, dealAnalysis, riskAssessment, recommendation };
}
```

### 2.2 WebSocket Server for Real-Time Streaming (`websocket/orchestrationEvents.ts`)

**Purpose**: Stream execution events to connected debugger clients in real-time.

**Implementation:**

```typescript
// websocket/orchestrationEvents.ts
import WebSocket from 'ws';
import { OrchestrationTracer } from '../services/orchestrationTracing';

const debuggerClients = new Set<WebSocket>();

export function setupOrchestrationWebSocket(server: any) {
  const wss = new WebSocket.Server({ server, path: '/ws/orchestration' });

  wss.on('connection', (ws: WebSocket, req: any) => {
    // Verify debugger access (dev-only or admin-only)
    if (!isDebuggerAccessAllowed(req)) {
      ws.close(4003, 'Unauthorized');
      return;
    }

    debuggerClients.add(ws);
    console.log(`Debugger connected. Clients: ${debuggerClients.size}`);

    // Subscribe to orchestration events
    const unsubscribe = OrchestrationTracer.subscribe((event) => {
      if (ws.readyState === WebSocket.OPEN) {
        ws.send(JSON.stringify({
          type: 'orchestration_event',
          data: event,
        }));
      }
    });

    ws.on('close', () => {
      debuggerClients.delete(ws);
      unsubscribe();
    });

    ws.on('message', async (message: string) => {
      try {
        const { action, payload } = JSON.parse(message);

        if (action === 'replay_agent') {
          // Handle prompt replay (see section 2.3)
          await handleReplayAgent(payload, ws);
        }
      } catch (error) {
        console.error('WebSocket message error:', error);
      }
    });
  });

  return wss;
}

function isDebuggerAccessAllowed(req: any): boolean {
  // In development: always allow
  if (process.env.NODE_ENV === 'development') return true;

  // In production: check admin role (implement based on your auth)
  const user = req.user; // Assumes authentication middleware sets this
  return user?.role === 'super_admin' || user?.role === 'admin';
}

export function broadcastToDebuggers(event: any) {
  debuggerClients.forEach(client => {
    if (client.readyState === WebSocket.OPEN) {
      client.send(JSON.stringify(event));
    }
  });
}
```

### 2.3 Prompt Replay Endpoint

**Purpose**: Allow isolated agent re-runs or cascade replays with modified prompts.

```typescript
// routes/debugger.ts (new file)
import express from 'express';
import { Orchestrator } from '../services/orchestrator';

const router = express.Router();

router.post('/api/debug/replay-agent', async (req, res) => {
  const { agentName, input, customPrompt, replayMode } = req.body;

  // replayMode: 'isolated' (just this agent) | 'cascade' (this agent + downstream agents)

  try {
    let result;

    if (replayMode === 'isolated') {
      // Run just this single agent with custom prompt
      result = await replayIsolatedAgent(agentName, input, customPrompt);
    } else if (replayMode === 'cascade') {
      // Run this agent + all downstream agents with custom prompt
      result = await replayCascadeAgents(agentName, input, customPrompt);
    }

    res.json({ success: true, result });
  } catch (error) {
    res.status(500).json({ success: false, error: error.message });
  }
});

async function replayIsolatedAgent(
  agentName: string,
  input: Record<string, any>,
  customPrompt: string
): Promise<any> {
  // Create temporary orchestrator with custom prompt
  const orchestrator = new Orchestrator();

  // Override the agent's prompt temporarily
  orchestrator.setCustomPrompt(agentName, customPrompt);

  // Call just that agent
  const result = await orchestrator.callAgent(agentName, input);

  return result;
}

async function replayCascadeAgents(
  startAgentName: string,
  input: Record<string, any>,
  customPrompt: string
): Promise<any> {
  // Find starting agent index
  const agents = ['creditExtractor', 'dealAnalyzer', 'riskAssessor', 'recommendationEngine'];
  const startIndex = agents.indexOf(startAgentName);

  if (startIndex === -1) throw new Error(`Unknown agent: ${startAgentName}`);

  // Execute from that point onward with custom prompt for that agent
  const orchestrator = new Orchestrator();
  orchestrator.setCustomPrompt(startAgentName, customPrompt);

  return orchestrator.executeOrchestrationFrom(startIndex, input);
}

export default router;
```

```typescript
async extractCreditRules(policyDocument: string) {

  for await (const rule of this.creditExtractionAgent.streamRules(policyDocument)) {
```

## 3. Frontend Implementation

### 3.1 WebSocket Hook (`hooks/useOrchestrationEvents.ts`)

```typescript
// hooks/useOrchestrationEvents.ts
import { useEffect, useRef, useCallback } from 'react';
import { useQueryClient } from '@tanstack/react-query';

export function useOrchestrationEvents(onEvent?: (event: OrchestrationEvent) => void) {
  const wsRef = useRef<WebSocket | null>(null);
  const queryClient = useQueryClient();

  useEffect(() => {
    // Connect to orchestration WebSocket
    const protocol = window.location.protocol === 'https:' ? 'wss:' : 'ws:';
    wsRef.current = new WebSocket(`${protocol}//${window.location.host}/ws/orchestration`);

    wsRef.current.onopen = () => {
      console.log('Connected to orchestration debugger');
    };

    wsRef.current.onmessage = (event) => {
      try {
        const message = JSON.parse(event.data);
        if (message.type === 'orchestration_event') {
          onEvent?.(message.data);
        }
      } catch (error) {
        console.error('Failed to parse WebSocket message:', error);
      }
    };

    wsRef.current.onerror = (error) => {
      console.error('WebSocket error:', error);
    };

    return () => {
      wsRef.current?.close();
    };
  }, [onEvent]);

  const replayAgent = useCallback(
    async (agentName: string, input: any, customPrompt: string, replayMode: 'isolated' | 'cascade') => {
      try {
        const response = await fetch('/api/debug/replay-agent', {
          method: 'POST',
          headers: { 'Content-Type': 'application/json' },
          body: JSON.stringify({ agentName, input, customPrompt, replayMode }),
        });

        if (!response.ok) throw new Error('Replay failed');
        return await response.json();
      } catch (error) {
        console.error('Agent replay error:', error);
        throw error;
      }
    },
    []
  );

  return { replayAgent };
}
```

### 3.5 Prompt Editor (`components/AIOrchestrationDebugger/PromptEditor.tsx`)

```typescript
// components/AIOrchestrationDebugger/PromptEditor.tsx
import React, { useState } from 'react';
import { motion } from 'framer-motion';

interface PromptEditorProps {
  agentName: string;
  input: Record<string, any>;
  onReplay: (customPrompt: string, mode: 'isolated' | 'cascade') => void;
}

export const PromptEditor: React.FC<PromptEditorProps> = ({ agentName, input, onReplay }) => {
  const [customPrompt, setCustomPrompt] = useState('');
  const [replayMode, setReplayMode] = useState<'isolated' | 'cascade'>('isolated');
  const [isReplaying, setIsReplaying] = useState(false);

  const handleReplay = async () => {
    if (!customPrompt.trim()) return;
    setIsReplaying(true);
    try {
      await onReplay(customPrompt, replayMode);
    } finally {
      setIsReplaying(false);
    }
  };

  return (
    <motion.div
      initial={{ opacity: 0, height: 0 }}
      animate={{ opacity: 1, height: 'auto' }}
      exit={{ opacity: 0, height: 0 }}
      className="bg-slate-800 rounded p-3 border border-slate-700 space-y-2"
    >
      <p className="text-xs font-semibold text-slate-300 uppercase">Edit & Replay</p>

      {/* Mode Selection */}
      <div className="flex gap-2 mb-2">
        <label className="flex items-center gap-2 text-xs text-slate-400 cursor-pointer">
          <input
            type="radio"
            value="isolated"
            checked={replayMode === 'isolated'}
            onChange={(e) => setReplayMode(e.target.value as any)}
            className="w-3 h-3"
          />
          Isolated Agent
        </label>
        <label className="flex items-center gap-2 text-xs text-slate-400 cursor-pointer">
          <input
            type="radio"
            value="cascade"
            checked={replayMode === 'cascade'}
            onChange={(e) => setReplayMode(e.target.value as any)}
            className="w-3 h-3"
          />
          Cascade Downstream
        </label>
      </div>

      {/* Prompt Editor */}
      <textarea
        value={customPrompt}
        onChange={(e) => setCustomPrompt(e.target.value)}
        placeholder="Enter custom system prompt for testing..."
        className="w-full h-24 p-2 bg-slate-950 text-slate-300 text-xs border border-slate-600 rounded font-mono resize-none"
      />

      {/* Replay Button */}
      <button
        onClick={handleReplay}
        disabled={!customPrompt.trim() || isReplaying}
```

### 3.6 Credit Extraction Preview (`components/AIOrchestrationDebugger/CreditExtractionPreview.tsx`)

```typescript
// components/AIOrchestrationDebugger/CreditExtractionPreview.tsx
import React from 'react';
import { motion, AnimatePresence } from 'framer-motion';
import { Badge } from '@/components/ui/badge';

interface CreditExtractionPreviewProps {
  session: OrchestrationSession;
}

export const CreditExtractionPreview: React.FC<CreditExtractionPreviewProps> = ({ session }) => {
  // Extract credit extraction events
  const creditEvents = session.events.filter(e =>
    e.eventType === 'credit_rule_extracted' || e.eventType === 'credit_extraction_batch'
  );

  const allRules = creditEvents.flatMap(e => e.rules || []);
  const lastProgress = creditEvents[creditEvents.length - 1]?.progress;

  return (
    <div className="bg-slate-800 rounded p-3 border border-slate-700 space-y-2">
      <p className="text-xs font-semibold text-slate-300 uppercase">
        Credit Extraction Preview
      </p>

      {/* Progress Bar */}
      {lastProgress && (
        <div className="space-y-1">
          <div className="flex justify-between text-xs text-slate-400">
            <span>Progress</span>
            <span>{lastProgress.percentage.toFixed(0)}%</span>
          </div>
          <div className="w-full bg-slate-950 rounded h-2">
            <motion.div
              initial={{ width: 0 }}
              animate={{ width: `${lastProgress.percentage}%` }}
              className="h-full bg-green-600 rounded transition-all"
            />
          </div>
        </div>
      )}

      {/* Rules List */}
      <div className="max-h-40 overflow-y-auto space-y-1">
        <AnimatePresence>
          {allRules.map((rule, idx) => (
            <motion.div
              key={`${rule.id}-${idx}`}
              initial={{ opacity: 0, x: -10 }}
              animate={{ opacity: 1, x: 0 }}
              className="bg-slate-950 p-2 rounded text-xs border border-slate-700"
            >
              <div className="flex justify-between items-start mb-1">
                <span className="font-mono text-cyan-400">Rule #{idx + 1}</span>
                <Badge variant="outline" className="text-xs">
                  {(rule.confidence * 100).toFixed(0)}%
                </Badge>
              </div>
              <p className="text-slate-300 font-medium mb-1">{rule.rule}</p>
              <p className="text-slate-500 text-xs italic">{rule.reasoning}</p>
            </motion.div>
          ))}
        </AnimatePresence>
      </div>

      <p className="text-xs text-slate-400">
        {allRules.length} rules extracted
      </p>
    </div>
  );
};
```

## 5. Usage Flow

### Credit Extraction Preview
1. User uploads credit policy document
2. Backend `creditExtractionAgent` processes policy
3. For each extracted rule, emit `OrchestrationEvent` with rule data + reasoning
4. Frontend receives events and renders in `CreditExtractionPreview` component
5. As stream completes, show final count and allow reviewing individual rules

### Real-Time Agent Orchestration
1. User triggers AI orchestration workflow
2. Backend starts session with `OrchestrationTracer.startSession()`
3. Each agent wrapped with `traceAgent()`, emits events for start/processing/complete
4. WebSocket streams events to frontend debugger in real-time
5. Developer sees:
   - Agent 1 running → output → complete
   - → switches to Agent 2 → output → complete
   - → switches to Agent 3 → output → complete
   - → switches to Agent 4 → final recommendation

### Prompt Editing & Replay
1. Developer clicks on agent in sidebar
2. `PromptEditor` component opens with current system prompt
3. Developer modifies prompt
4. Developer selects mode:
   - **Isolated**: Re-run just Agent 2 with new prompt, same input as before
   - **Cascade**: Re-run Agent 2 + 3 + 4 with new Agent 2 prompt, using Agent 1's output
5. Backend replays via `POST /api/debug/replay-agent`
6. New execution traced and streamed to debugger
7. Developer compares side-by-side with previous run

## 6. Development Considerations

### Performance
- Stream large outputs incrementally (don't send entire 100KB response at once)
- Batch credit extraction rules (emit every 5-10 rules, not every single rule)
- Limit event history to last 50 events per session to avoid memory issues
- Consider storing detailed sessions in browser IndexedDB for longer history

### Security
- Restrict debugger access to dev environment OR admin users only
- Don't expose full API keys or sensitive data in event streams
- Consider redacting PII from displayed outputs
- Implement CSRF protection for replay endpoint

### UX Enhancements (Phase 2)
- Export session as JSON for sharing
- Diff tool to compare prompt variations side-by-side
- Save favorite prompts and compare results
- Integration with GitHub to save optimized prompts as code commits
- Replay with historical data (run Agent 3 with Agent 1/2 outputs from a past session)
