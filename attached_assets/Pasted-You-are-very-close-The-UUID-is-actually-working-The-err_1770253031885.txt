## 🧠 How PandaDoc Actually Works (Important Concept)

Inside PandaDoc templates, each signature participant is assigned a **ROLE**.

Example:

| Template Role | Meaning            |
| ------------- | ------------------ |
| Borrower      | person signing LOI |
| Lender        | internal signer    |
| Guarantor     | optional signer    |

When your API creates a document, you must assign recipients to those exact roles.
