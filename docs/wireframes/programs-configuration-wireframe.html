Programs Configuration — Comprehensive Wireframe v2

  • 3-Level Progressive Disclosure: L1 = Summary strip + scannable table. L2 = Expandable rows with metrics, property types, quick actions. L3 = Slide-over panel (760px) with 5 tabs for full program editing.
  • Slide-over panel has 5 tabs: Details, Workflow, Documents, Pricing, AI Rules — replacing the old modal with context-preserving drawer.
  • Creation Wizard: 9-step sidebar wizard with template selection first. Non-linear navigation. Steps: Template → Credit Policy → Details → Quote Form → Stages → Documents → Tasks → Pricing → Review.
  • Financial tooltips: LTV, DSCR, YSP, Points, NOI, Base Rate all show plain-English explanations on hover (dashed underline affordance).
  • Auto-save: Green indicator in panel header replaces manual Save button. Every field change persists automatically.
  • Workflow timeline: Vertical stage visualization with doc/task counts per stage, drag-to-reorder hint, color-coded stage dots.
  • Document management: Categorized (Borrower, Entity, Property, Financial, Closing), with required/optional badges, visibility controls (Admin/Broker/All), stage assignment.
  • Pricing engine: 4 modes — None, Rule-based (base rate + FICO/LTV/property/amount/DSCR/purpose adjusters + disqualifiers), AI Upload, External URL. YSP and Points configurable with broker toggle.
  • AI Review Rules: Voice-to-rule recording, condition builder, severity levels. Integrates credit policy rules.
  • Empty states: Meaningful CTAs when no programs/docs/tasks exist. Never a blank screen.

AI Review Rules

Automated checks applied during deal processing. 6 active rules.

Voice-to-Rule
Describe a rule in plain English and AI will convert it to a structured review rule. Example: "Flag any deal where LTV exceeds 75% and DSCR is below 1.25"
🔴 High LTV + Low DSCR Flag
Critical
IF ltv > 75% AND dscr < 1.25 THEN flag("Review Required: High-risk LTV/DSCR combination")
🟡 Low FICO Warning
Warning
IF fico < 700 THEN warn("Below-average FICO score — manual underwriting recommended")
🔴 Max Loan Amount Exceeded
Critical
IF loan_amount > 2500000 THEN reject("Loan amount exceeds program maximum of $2,500,000")
🟡 Cash-Out Refi LTV Cap
Warning
IF loan_purpose == "cash_out" AND ltv > 70% THEN flag("Cash-out refi LTV exceeds 70% guideline")
🟢 Missing Rent Roll Check
Info
IF property_type IN ["2-4 Unit", "Multi-Family"] AND !has_document("Rent Roll") THEN remind("Rent roll required for multi-unit properties")
🟡 Foreign Entity Review
Warning
IF entity_type == "foreign" THEN flag("Foreign entity — additional compliance review required")
